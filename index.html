BishBosh: the number checker

BishBosh is a program that checks what numbers can be divided by selected divisors. You input a number, divisor A and divisor B. The program will run through all numbers from 0 to your input and if divisible by one of the entered divisors exclaim either Bish! or Bosh!. If a number is divisibly by both numbers it will exclaim BishBosh!.
